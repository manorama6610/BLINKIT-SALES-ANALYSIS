## 📊 Blinkit Sales Performance Dashboard (Power BI)

### 🔍 Project Overview
An interactive Power BI dashboard built to analyze the sales and operational performance of a retail grocery business using a Blinkit-style dataset. The dashboard provides actionable insights into outlet performance, product categories, and customer behavior through intuitive visualizations.

---

### 🗂️ About the Dataset
- **Source:** Microsoft official sample Excel workbook  
- **Type:** Simulated retail grocery sales data  
- **Format:** Excel (.xlsx)

The dataset includes product details, outlet attributes, sales metrics, and customer ratings.  
*This data is used solely for learning and portfolio demonstration purposes and does not represent real Blinkit business data.*

---

### 🎯 Problem Statement
The objective of this project is to analyze sales trends and key performance metrics across different outlet locations, sizes, and product categories, enabling stakeholders to make data-driven decisions.

---

### 📌 Key Insights
- Tier-2 outlets generate the highest sales despite fewer items.
- Medium-sized outlets outperform small outlets in overall revenue.
- Low-fat products contribute higher sales in Tier-1 locations.
- Sales peaked around 2018, reflecting outlet expansion impact.
